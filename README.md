# 3D Event Celebration Page

## Description

This project is an interactive web page created as a practical exercise within a Full Stack Developer course, showcasing the capabilities of 3D graphics
and animation in web development. Utilizing HTML for structure, CSS for styling and animations, and JavaScript for dynamic elements, the project conjures
a festive atmosphere with animated snowflakes and greetings.

## Installation

Clone the repository using:

git clone https://github.com/Maysker/3devent.git

## Usage

Simply open the 3devent.html file in a browser to enjoy the festive theme and animation.
